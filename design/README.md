# UI Design Sketches

These sketches represent the initial design thinking for the LMS prototype.

## Key Screens

- Login & Registration: Simple authentication flow
- Course Catalog: Displays available courses
- Course Details: Shows lessons and progress tracking
- Admin Panel: Allows course creation and audit log viewing

## Design Approach

The UI was designed with simplicity and clarity in mind:
- Minimal navigation
- Clear separation of roles (Admin vs Learner)
- Easy access to key actions (view courses, complete lessons)

These sketches guided the implementation of the final UI.
